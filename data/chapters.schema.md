# chapters.json — Data Contract

Generated by Python pipeline (`parser.py`), consumed by Vue `ReaderView.vue`.

## Schema

```json
{
  "bookId": "string — unique slug (e.g. 'the-giver')",
  "title": "string — book title",
  "author": "string — author name",
  "chapters": [
    {
      "id": "string — chapter slug (e.g. 'ch-01')",
      "title": "string — chapter title",
      "paragraphs": [
        {
          "id": "string — paragraph slug (e.g. 'p-01-001')",
          "text": "string — paragraph text (clean, no HTML)",
          "annotatedWords": ["string — words with dictionary entries in this paragraph"]
        }
      ]
    }
  ]
}
```
