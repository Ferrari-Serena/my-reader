# dictionary.json — Data Contract

Generated by Python pipeline (`dictionary.py`), consumed by Vue `WordPopup.vue`.

## Schema

```json
{
  "bookId": "string — matches chapters.json",
  "words": {
    "<lemma>": {
      "lemma": "string — base form",
      "phonetic": "string — pronunciation (MW format or empty)",
      "partOfSpeech": "string — e.g. 'noun', 'verb', 'adjective'",
      "definitions": ["string — up to 5 short definitions"],
      "audioUrl": "string — MW pronunciation MP3 URL (or empty)",
      "level": "string | null — 'SAT', 'AP', or null",
      "chapters": ["string — chapter IDs where this word appears"]
    }
  }
}
```

See `reader/public/books/the-giver/dictionary.json` after running the generator.
